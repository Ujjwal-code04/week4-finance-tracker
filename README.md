## About  

Personal Finance Tracker is a console-based Python application that helps users record, organize, and analyze their daily expenses.  
It uses file handling to store data permanently and generate useful financial reports.

---

## Features  

- Add and view expenses  
- Categorize spending (Food, Transport, Bills, etc.)  
- Save data in JSON file  
- Automatic backup system  
- Monthly and category-wise reports  
- Clean menu-driven interface
